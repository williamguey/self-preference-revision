SUBSTRATE:
  Reused the exact v2 substrate = all 63 len-2-3 all-editable-whitelist prompts

FRESH-DECIDER ROTATION:
  author = the family that actually generated the draft. fresh = a different family,
  rotated across drafts (candidates = families minus author; fresh =
  candidates[usable_index % 3]) so every family serves as author and as fresh.
  Recorded per draft.

ONE GOOD FIX: first checker-verified GOOD targeted fix per draft (generation order),
  same rule as v2. Fixer model = gpt-4o-mini (cheap), unchanged from v2.

CI / UNIT OF ANALYSIS: draft is the unit. per-draft author_reject_rate and
  fresh_reject_rate (means of 3 reps); gap_i = author_reject - fresh_reject;
  GAP = mean(gap_i); 95% CI = paired bootstrap over drafts (10000 resamples,
  seed=12345). Same method as v2.

PAIRED SKEPTIC/MIRROR COUNTS: reported with the v2-comparable "any-rep" definition
  (skeptic = author REJECT in >=1 rep AND fresh ACCEPT in >=1 rep; mirror = the
  reverse) AND a majority-vote definition as robustness.

QUALITATIVE CLASSIFICATION (stricter-than-checker vs preference): each author
  rejection reason is classified by gpt-4o-mini into STRICTER / PREFERENCE / OTHER
  with a fixed rubric. All raw reasons are also saved for manual review. (An LLM
  classifier is used because the distinction is semantic; this is an analysis step
  the spec explicitly requests, not manufactured data.)

  pool (same whitelist + no_comma), using prompts NOT already used, and appends
